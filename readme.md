# Arduino Projects

Hardware projects built on the Arduino platform, covering sensor interfacing and display output.

## Projects

**PresenceSensorWithLEDandLCD** — Detects presence and displays status on an LCD with LED feedback.

**WeatherStation** — Reads temperature and humidity from a DHT sensor and outputs readings over serial.

**WeatherStationWithLCD** — Same as above, with readings displayed on a 16x2 LCD screen.

**WeatherStationWithBlynk** — Weather station connected to the Blynk mobile app for remote monitoring over Wi-Fi. Requires an ESP8266 or compatible board.

## Requirements

- Arduino IDE
- Libraries (install via Library Manager): `LiquidCrystal`, `DHT sensor library`, `Blynk`
- Hardware varies per project — refer to each project folder for wiring details
